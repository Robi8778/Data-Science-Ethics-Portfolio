# Data-Science-Ethics-Portfolio
Diese Analyse zeigt das Phänomen des Fairness Gerrymandering: Während das Unternehmen statistisch nachweisen kann, dass sowohl Frauen als auch People of Color eingestellt werden, zeigt die intersektionale Analyse, dass Frauen of Color systematisch ausgeschlossen wurden. Dies beweist, dass isolierte Metriken Diskriminierung verstecken können.

Wann ist ein Algorithmus wirklich fair? ⚖️🤖

Ich habe mich vor kurzem mit dem Thema "Fairness Gerrymandering" beschäftigt. In einer simulierten Fallstudie habe ich Einstellungsdaten analysiert, um Diskriminierung auf die Spur zu kommen.

Das Spannende: Ein Unternehmen kann statistisch belegen, dass es "genug Frauen" und "genug People of Color" einstellt – und trotzdem eine bestimmte Gruppe (wie Frauen in der Kategorie "People of Color") systematisch ausschließen.

Meine Erkenntnisse aus der Analyse:
📍 Daten lügen nicht, aber sie können die halbe Wahrheit verbergen.
📍 Intersektionalität muss in jedem Bias-Check berücksichtigt werden.
📍 Als Data Scientists tragen wir die Verantwortung, tiefer zu graben als nur bis zur ersten Oberfläche der Statistik.
